# Proxy SwitchyOmega

SwitchyOmega 是 chrome 的一个插件，可以使谷歌浏览器访问指向一个代理。

可以在商店安装该插件，安装之后浏览器右上角会出现圆环的图标

点击圆环->选项可以进入设置页面，按照教程指引即可。
